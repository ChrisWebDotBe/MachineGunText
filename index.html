Chrisweb.be - MachineGunText

Un plugin sympa pour animer du texte comme une mitraillette...

Ce plugin nécessite jQuery et GreenSock TweenMax

Téléchargement

jQuery : ici

TweenMax : ici

Le code source sur GitHub : MachineGunText


Utilisation

<html>
<head>
	<script type="text/javascript" src="js/jquery-1.10.1.min.js"></script>
	<script type="text/javascript" src="js/TweenMax.min.js"></script>
	<script type="text/javascript" src="js/MachineGunText.min.js"></script>
</head>
<body>
<script>
	$(document).ready(function() {
	  var options={
		text:"Voici le résultat de ce plugin ! Ainsi que tous les paramètres disponibles...",
		height: '100px'
	  }
	  $('#demo1').MachineGunText(options);
	});
</script>
</body>
</html>

Paramètres

NomPar défautExplication

width'100%'Largeur du conteneur

height'50px'Hauteur du conteneur

bgcolor'#000'Couleur de fond

bgimage''Image de fond (facultative)

bgrepeat'no-repeat'Répétition de l'image de fond

bgposition'top center'Position de l'image de fond

bgattachment'fixed'Attachement de l'image de fond

bgsize'cover'Dimension de l'image de fond (cover,contain,auto)

text'votre texte'Le texte à faire défiler. Peut être une chaine ou un tableau, le tableau servant à définir où il faut couper le texte (fin à chaque .?!)

Par exemple : ['Phrase 1','Phrase 2','Dernière phrase']

maxlenght12Nombre de caractères maximum pour couper le texte

fontcolor'#fff'Couleur du texte

fontsize'50px'Hauteur du texte

fontfamily"Arial"Police de caractère

fontweight'400'Grosseur de la police

animsentencemaxduration0.5Temps maximum pour chaque phrase

animdelay0.6Temps en seconde avant de lancer l'animation

animrepeat-1Nombre de répétitions (-1 : à l'infini, 0 : aucune)

animrepeatdelay2Temps en seconde avant chaque répétition

animdramaticpausetrueActive une pause entre chaque phrase

animdramaticpauseduration0.6Temps en seconde de la pause

animmaxscale1.2Echelle maximum du texte grossi

callbacknullFonction à appeler à la fin de l'animation

callbacksentencenullFonction à appeler à la fin de chaque phrase


CSS

Pour mettre en forme le conteneur ou chaque phrase, 2 classes sont disponibles :

<style>
.MachineGunTextContainer {

	margin-bottom:15px;

Exemples

Une version avec tous les paramètres :

<script>
$(document).ready(function() {
  var cpt=0;
  var options={
	width: '800px',
	height: '350px',
	bgcolor: '#FE1',
	bgimage: '',
	bgrepeat: 'no-repeat',
	bgposition: 'top center',
	bgattachment: 'fixed',
	bgsize: 'cover',
	text: "don't forget to set text",
	maxlenght: 12,
	fontcolor: '#412',
	fontsize: '70px',
	fontfamily: "Comic Sans MS",
	fontweight: '700',
	animsentencemaxduration: 1.5,
	animdelay: 0.2,
	animrepeat: 1,
	animrepeatdelay: 1,
	animdramaticpause: true,
	animdramaticpauseduration: 0.9,
	animmaxscale: 2.2,
	callback: function() {alert('fin après '+cpt+' phrases');},
	callbacksentence: function () {cpt++;}
	}; 
  }
  $('#votreconteneur').MachineGunText(options);
});
</script>
